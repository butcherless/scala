# Learning Kafka Producer / Consumer

## Documentation

https://kafka.apache.org/documentation/

## Commands

start services

- docker-compose -f ./single-broker.yml up -d

stop services

- docker-compose -f ./single-broker.yml stop

show logs
- docker-compose -f ./single-broker.yml logs -f
